Capa de Captura — Punto de Venta (POS)
🏪 Tienda Física
POS + Escáner QR

El barista escanea el QR del cliente al momento del pago. El POS registra el monto de la transacción, el producto y la sucursal. Única vía válida para acumular y canjear puntos.

Acumulación Canje Solo presencial
📡 Integración
API Gateway / Webhook

Cada transacción del POS dispara un evento en tiempo real hacia la plataforma Piggy. Incluye: monto GTQ, timestamp, sucursal, cliente_id y tipo de operación (compra / canje).

REST API Webhook Real-time

⚙️ Motor de Reglas
Engine de Puntos

Aplica la regla Q1 = 1 punto. Valida tope de 4,000 pts en saldo activo. Registra historial acumulado independiente del saldo (para cálculo de nivel).

Level Manager

Silver → Gold (>1,500 hist.) → Platinum (>3,500 hist.). El nivel se basa en historial, no en saldo. Reinicio anual de niveles cada enero.

Silver Gold Platinum
